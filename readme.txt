== Description ==

Adds a a new "All" tab in the BuddyPRess Members Profile and makes it the default landing tab in order to create a Facebook like Wall.

Simply install and activate the plugin then visit Profile>>Activity.

== Installation ==

1. From the Admin>>Plugins>>Add New page, search for BP Profile Activity Wall.
2. When you have located the plugin, click on "Install" and then "Activate".
